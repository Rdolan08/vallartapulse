## 2\. If You Start Managing Properties Directly

If the company begins doing things like:

- listing properties for rent

- negotiating rental agreements

- representing owners

- collecting payments

- managing bookings

## 3\. If You Take Commissions on Rentals

Another gray area is **commission-based booking services**.

If you:

- advertise properties

- collect booking payments

- take commissions on rentals


then regulators may view that as **real estate brokerage or hospitality services**.

The risk is not huge, but it is safer to structure the business as:

• **software subscription**

• **vendor marketplace**

• **analytics platform**

rather than a booking agency.

Connecting owners with:

- cleaners

- electricians

import pandas as pd
import streamlit as st
import plotly.express as px

st.set_page_config(page_title="Puerto Vallarta Market Dashboard", layout="wide")

st.title("Puerto Vallarta Market Dashboard")
st.caption("Prototype for tourism and vacation rental intelligence")

@st.cache_data
def load_data():
    listings = pd.read_csv("data/listings.csv")
    flights = pd.read_csv("data/flights.csv")
    cruises = pd.read_csv("data/cruises.csv")
    return listings, flights, cruises

listings, flights, cruises = load_data()

neighborhoods = ["All"] + sorted(listings["neighborhood"].dropna().unique().tolist())
selected = st.selectbox("Neighborhood", neighborhoods)

filtered = listings.copy()
if selected != "All":
    filtered = filtered[filtered["neighborhood"] == selected]

avg_rate = round(filtered["nightly_rate"].mean(), 0)
avg_occ = round(filtered["occupancy_rate"].mean() * 100, 1)
avg_rev = round(filtered["monthly_revenue_est"].mean(), 0)
arrivals = int(flights["arrivals_next_7d"].sum() + cruises["passengers_next_7d"].sum())

c1, c2, c3, c4 = st.columns(4)
c1.metric("Avg Nightly Rate", f"${avg_rate:,.0f}")
c2.metric("Estimated Occupancy", f"{avg_occ}%")
c3.metric("Estimated Monthly Revenue", f"${avg_rev:,.0f}")
c4.metric("Demand Signal (7d)", f"{arrivals:,}")

st.subheader("Nightly Rate Trend")
trend = filtered.groupby("month", as_index=False)["nightly_rate"].mean()
fig1 = px.line(trend, x="month", y="nightly_rate", markers=True)
st.plotly_chart(fig1, use_container_width=True)

st.subheader("Neighborhood Comparison")
comp = listings.groupby("neighborhood", as_index=False)[["nightly_rate", "occupancy_rate"]].mean()
fig2 = px.bar(comp.sort_values("nightly_rate", ascending=False), x="neighborhood", y="nightly_rate")
st.plotly_chart(fig2, use_container_width=True)

st.subheader("Top Areas This Month")
table = listings.groupby("neighborhood", as_index=False).agg(
    avg_nightly_rate=("nightly_rate", "mean"),
    avg_occupancy=("occupancy_rate", "mean"),
    avg_monthly_revenue=("monthly_revenue_est", "mean")
).sort_values("avg_monthly_revenue", ascending=False)
st.dataframe(table, use_container_width=True)

st.subheader("Investor Quick Calculator")
col1, col2, col3 = st.columns(3)
purchase_price = col1.number_input("Purchase Price", value=450000)
hoa_monthly = col2.number_input("HOA / Monthly Fixed Costs", value=600)
mgmt_pct = col3.slider("Management %", min_value=10, max_value=40, value=20)

nightly_rate = st.number_input("Expected Nightly Rate", value=int(avg_rate if avg_rate == avg_rate else 180))
occ_pct = st.slider("Expected Occupancy %", min_value=20, max_value=95, value=int(avg_occ if avg_occ == avg_occ else 65))

gross_annual = nightly_rate * 365 * (occ_pct / 100)
management_cost = gross_annual * (mgmt_pct / 100)
fixed_costs = hoa_monthly * 12
net_before_tax = gross_annual - management_cost - fixed_costs
yield_pct = (net_before_tax / purchase_price) * 100 if purchase_price else 0

a, b, c = st.columns(3)
a.metric("Estimated Annual Gross", f"${gross_annual:,.0f}")
b.metric("Net Before Tax", f"${net_before_tax:,.0f}")
c.metric("Simple Yield", f"{yield_pct:.2f}%")
